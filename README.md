# 💔 PDF Broke My Heart

PDF Broke My Heart is an open-source PDF toolkit that helps users perform common PDF operations easily in the browser. The goal of this project is to build a free and community-driven alternative to tools like iLovePDF.

Users will be able to merge, compress, convert, and manage PDF files through a fast and modern interface.

This project is built with modern web technologies and is open for contributions from developers.

---

# 🚀 Features (Planned)

- Merge PDF
- Split PDF
- Compress PDF
- Image to PDF
- PDF to Image
- Rotate PDF
- Add watermark
- Remove pages
- Protect / Unlock PDF

More tools will be added with community contributions.

---

# 🛠 Tech Stack

- **Next.js**
- **React**
- **Tailwind CSS**
- **Shadcn UI**
- **TypeScript**
- **Node.js**
